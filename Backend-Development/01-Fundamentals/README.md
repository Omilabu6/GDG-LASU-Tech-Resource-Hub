# Backend Development Fundamentals

This section introduces the core ideas behind backend development.

Before building backend systems, it's important to understand how the web works and how servers process requests.

---

# 1. How the Internet Works

Understanding how data travels between browsers and servers is essential.

## Concepts to Learn

- client vs server
- HTTP and HTTPS
- requests and responses
- DNS
- hosting

## Resource

MDN – How the Web Works  
https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/How_the_Web_works

---

# 2. What is a Server?

A server is a computer that processes requests and returns responses.

For example:

- a browser sends a request
- the server processes it
- the server returns data

## Resource

Introduction to Web Servers  
https://developer.mozilla.org/en-US/docs/Learn/Common_questions/What_is_a_web_server

---

# 3. Introduction to Node.js

Node.js allows developers to run JavaScript on the server.

It is widely used for backend development.

## Concepts to Learn

- running JavaScript outside the browser
- Node modules
- package management (npm)

## Resource

Node.js Official Docs  
https://nodejs.org/en/docs

# 4. Introduction to Express.js

Express is a popular framework used with Node.js for building APIs.

## What to Learn

- creating a server

- sending responses
