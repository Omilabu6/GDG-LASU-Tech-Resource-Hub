# Core Concepts

This section introduces the key systems used in backend development.

---

# 1. REST APIs

APIs allow applications to communicate with each other.

REST APIs are commonly used in web development.

## What to Learn

- GET requests
- POST requests
- PUT requests
- DELETE requests

## Resource

REST API Guide  
https://restfulapi.net/

---

# 2. Working with Databases

Databases store application data.

Common databases include:

- MySQL
- PostgreSQL
- MongoDB

## Concepts

- storing data
- retrieving data
- updating records

## Resource

MongoDB University  
https://university.mongodb.com/

---

# 3. Backend Routing

Routes define how servers respond to requests.

Example routes:
GET /users
POST /users
GET /products


These routes return different data from the server.

---

# 4. Middleware

Middleware functions run between the request and response.

They are used for:

- logging
- authentication
- validation

## Resource

Express Middleware Guide  
https://expressjs.com/en/guide/using-middleware.html

---

# 5. Error Handling

Servers must handle errors gracefully.

Examples include:

- invalid requests
- server crashes
- missing data

Learning proper error handling improves application stability.
