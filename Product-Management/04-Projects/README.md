# Product Management Projects

Projects help build a portfolio of product thinking and decision-making skills.

---

# Beginner Projects

Product Improvement Case Study

Choose an existing app and propose improvements.

Examples:

- improve onboarding flow
- simplify navigation
- improve search functionality

---

# Feature Proposal

Design a new feature for an existing product.

Example:

Adding a collaborative feature to a note-taking app.

Include:

- problem definition
- proposed solution
- expected impact

---

# Intermediate Projects

Product Strategy Case Study

Create a strategy for launching a new digital product.

Include:

- target users
- product goals
- feature roadmap
- success metrics
