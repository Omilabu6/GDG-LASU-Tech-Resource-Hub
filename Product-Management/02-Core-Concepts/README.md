# Core Concepts

This section introduces important frameworks and concepts used in product management.

---

# 1. Product Discovery

Product discovery focuses on identifying problems worth solving.

Activities include:

- user research
- problem validation
- testing product ideas

Resource

Product Discovery Guide  
https://www.svpg.com/product-discovery/

---

# 2. Product Roadmaps

A product roadmap outlines the direction and priorities for a product.

It helps teams understand:

- upcoming features
- development priorities
- long-term product vision

---

# 3. Prioritization Frameworks

Product managers must decide which features to build first.

Common frameworks include:

- RICE (Reach, Impact, Confidence, Effort)
- MoSCoW prioritization
- Value vs Effort analysis

---

# 4. Product Metrics

Product managers track metrics to evaluate product success.

Examples include:

- user growth
- engagement
- retention
- churn
