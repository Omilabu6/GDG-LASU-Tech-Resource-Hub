# Game Development Fundamentals

This section introduces the basic ideas behind game development.

Game development combines **programming, design, art, and problem solving** to create interactive experiences.

Before building complex games, it is important to understand how game engines work and how basic gameplay systems are created.

---

# 1. What is Game Development?

Game development is the process of creating games for platforms such as:

- PC
- Mobile
- Web
- Consoles

Games are built using **game engines**, which provide tools for graphics, physics, sound, and scripting.

# 3. Introduction to Game Engines

Game engines provide tools for building games more efficiently.

They handle things like:

- rendering graphics
- physics simulations
- scene management
- scripting

Two beginner-friendly engines are:

Unity  
https://unity.com

Godot  
https://godotengine.org
