# Intermediate Game Development

This section explores deeper game development systems used in more complex games.

By this stage, learners should already understand:

- basic programming
- player movement
- collision detection
- simple game mechanics
- basic game projects

Now the focus shifts to building **richer gameplay systems and better player experiences**.

---

# 1. Game AI

Game AI controls the behavior of enemies or non-player characters (NPCs).

Examples include:

- enemy chasing the player
- patrolling guards
- enemies attacking the player

## Concepts to Learn

- state machines
- pathfinding
- enemy behavior patterns

## Resources

Game AI Basics  
https://gameprogrammingpatterns.com/state.html

Unity AI Navigation  
https://docs.unity3d.com/Manual/nav-Overview.html

Godot Navigation  
https://docs.godotengine.org/en/stable/tutorials/navigation/index.html

# 3. Level Design

Level design focuses on creating environments where gameplay happens.

Good level design improves player experience and game progression.

## What to Learn

- level layout
- player guidance
- difficulty progression
- environment design

## Resource

Introduction to Level Design  
https://www.gamedeveloper.com/design/an-introduction-to-level-design
