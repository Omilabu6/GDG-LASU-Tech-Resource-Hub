# Projects

Projects are the best way to improve your game development skills.

This section provides project ideas that help learners apply what they learned from:

- Fundamentals
- Core Concepts
- Practice

Try building these games step by step. Start with simple projects and gradually move to more complex ones.

---

# Beginner Projects

These projects help you understand basic gameplay mechanics.

## Pong Clone

Pong is one of the simplest games to build.

Features to implement:

- paddle movement
- ball movement
- collision detection
- score tracking

Skills practiced:

- player input
- collision systems
- game loops

---

## Flappy Bird Clone

This project helps you understand physics and obstacle mechanics.

Features:

- player jumping mechanic
- gravity
- obstacle spawning
- score system

Skills practiced:

- physics
- timing systems
- game state management

---

## Brick Breaker

Create a simple brick-breaking game.

Features:

- paddle control
- bouncing ball
- brick destruction
- score counter

Skills practiced:

- collision detection
- object interaction
- simple physics

---

# Intermediate Projects

These projects introduce more complex gameplay systems.

## Endless Runner

Create a game where the player runs continuously while avoiding obstacles.

Features:

- character movement
- obstacle spawning
- increasing difficulty
- score tracking

Skills practiced:

- procedural spawning
- movement systems
- game balancing

---

## Platformer Game

A platformer is a classic game genre.

Features:

- player movement
- jumping mechanics
- platforms
- enemy movement
- collectibles

Skills practiced:

- physics systems
- level design
- collision detection

---

## Top-Down Shooter

Build a simple top-down shooting game.

Features:

- player movement
- shooting mechanics
- enemy spawning
- health system

Skills practiced:

- input systems

- projectile systems

After completing a project, consider publishing it online.

Recommended platforms:

Itch.io  
https://itch.io

Game Jolt  
https://gamejolt.com

Publishing your game helps build a **portfolio and showcase your work**.

---

# Project Tips

When building games:

- start small
- focus on core mechanics
- test your game frequently
- improve visuals later
- share your progress

Game development improves through **iteration and experimentation**.
