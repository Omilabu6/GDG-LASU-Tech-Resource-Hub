# Advanced Game Development

This section explores advanced topics used in professional game development.

These concepts help improve performance, scalability, and complexity in games.

Optimizing a game ensures it runs smoothly on different devices.

- performance profiling
- reducing draw calls
- optimizing scripts
- managing assets

Multiplayer games allow players to interact with each other in real time.

Examples include:

- cooperative gameplay
- online battles
- competitive multiplayer

## Concepts

- networking
- synchronization
- client-server architecture

## Resources
