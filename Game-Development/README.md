# Game Development

Game development involves designing, programming, and creating interactive experiences for players.

This learning track provides resources and guidance for anyone interested in building games, from beginners to intermediate learners.

The goal is to help learners understand the **core concepts of game development** and build playable games.

---

# Who This Track Is For

This track is suitable for:

- Beginners who want to learn how games are made
- Developers interested in interactive programming
- Artists and designers curious about game engines
- Anyone who enjoys building games

---

# How to Use This Track

Follow the folders in order:

1. **01-Fundamentals** → learn basic programming and game engine basics
2. **02-Core-Concepts** → understand game mechanics and logic
3. **03-Practice** → small exercises to improve skills
4. **04-Projects** → build playable games
5. **05-Intermediate** → deeper game development concepts
6. **06-Advanced** → optimization and professional workflows

---

# Game Engines Covered

This track focuses primarily on beginner-friendly engines:

Unity  
https://unity.com

Godot  
https://godotengine.org

These engines are widely used and well documented.

---

# Programming Languages

Common languages used in game development include:

- C#
- GDScript
- C++
- JavaScript (for web games)

In this track we mainly focus on **C# (Unity)** and **GDScript (Godot)** because they are beginner friendly.

---

# What You Will Learn

By following this track you will learn how to:

- Understand how game engines work
- Create game scenes
- Implement player movement
- Detect collisions
- Manage game states
- Build simple playable games

---

# Learning Approach

The best way to learn game development is to **build games**.

Expect to:

- experiment frequently
- make mistakes
- iterate on ideas
- improve gradually

Start small and gradually build more complex games.
