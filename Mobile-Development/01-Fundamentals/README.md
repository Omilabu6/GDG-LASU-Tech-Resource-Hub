# Mobile Development Fundamentals

This section introduces the basic concepts needed before building mobile applications.

---

# 1. How Mobile Apps Work

Mobile apps are software applications designed to run on smartphones and tablets.

Apps interact with device features such as:

- camera
- GPS
- storage
- internet connection

Resource

Introduction to Mobile Development  
https://developer.android.com/guide

---

# 2. Native vs Cross-Platform Development

Mobile apps can be developed in two main ways.

Native Development  
Apps are built specifically for one platform.

Examples:

- Android apps built with Kotlin or Java
- iOS apps built with Swift

Cross-Platform Development  
Apps are built once and run on multiple platforms.

Examples:

- React Native
- Flutter
