# Core Concepts

This section introduces important systems used in mobile applications.

---

# 1. User Interface (UI)

Mobile apps rely heavily on intuitive user interfaces.

Important UI components include:

- buttons
- text fields
- images
- navigation bars

Resource

Android UI Guide  
https://developer.android.com/guide/topics/ui

Flutter UI Guide  
https://docs.flutter.dev/ui

---

# 2. Navigation

Mobile apps often have multiple screens.

Navigation allows users to move between screens.

Examples include:

- stack navigation
- tab navigation
- drawer navigation

---

# 3. State Management

Apps often need to manage dynamic data.

Examples include:

- user input
- fetched data
- application state

---

# 4. Working with APIs

Most mobile apps communicate with backend servers.

APIs allow apps to:

- retrieve data
- send user data
- synchronize information

---

# 5. Local Storage

Apps often store data locally on the device.

Examples include:

- user preferences
- cached data
- offline data storage
