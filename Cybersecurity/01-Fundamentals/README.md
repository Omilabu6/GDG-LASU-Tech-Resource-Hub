# Cybersecurity Fundamentals

This section introduces the core ideas behind cybersecurity.

Before learning advanced security techniques, it is important to understand how computer systems and networks operate.

---

# 1. What is Cybersecurity?

Cybersecurity refers to protecting computer systems, networks, and data from unauthorized access or attacks.

Common cybersecurity threats include:

- malware
- phishing
- ransomware
- data breaches

# 2. Types of Cyber Attacks

Understanding attacks helps security professionals defend against them.

Common attacks include:

- phishing attacks
- password attacks
- denial of service attacks
- malware infections

# 3. Networking Basics

Cybersecurity professionals must understand networking.

Important concepts include:

- IP addresses

- HTTP and HTTPS
- ports and protocols
