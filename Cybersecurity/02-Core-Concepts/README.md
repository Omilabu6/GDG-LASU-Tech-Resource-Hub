# Core Concepts

This section introduces important cybersecurity systems used to protect networks and applications.

---

# 1. Encryption

Encryption protects data by converting it into unreadable formats.

Examples include:

- HTTPS encryption
- password hashing
- secure communication

Resource

Encryption Basics  
https://www.cloudflare.com/learning/ssl/what-is-encryption/

---

# 2. Authentication

Authentication verifies the identity of users.

Common methods include:

- passwords
- multi-factor authentication
- biometrics

---

# 3. Authorization

Authorization controls what users are allowed to access after authentication.

Examples:

- admin access
- user permissions
- role-based access control

---

# 4. Firewalls

Firewalls monitor and control incoming and outgoing network traffic.

They protect systems from unauthorized access.

Resource

Firewall Basics  
https://www.cloudflare.com/learning/ddos/glossary/firewall/

---

# 5. Vulnerability Scanning

Security tools scan systems to detect vulnerabilities.

These tools help identify weaknesses before attackers exploit them.
