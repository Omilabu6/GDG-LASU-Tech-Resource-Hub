# Web Application Security

Many cyber attacks target websites.

Common vulnerabilities include:

- SQL injection
- cross-site scripting (XSS)
- cross-site request forgery (CSRF)

Network security protects communication systems.

# Incident Response

Security teams must respond quickly to attacks.

Incident response includes:

- detecting attacks
- analyzing the breach
- restoring systems
