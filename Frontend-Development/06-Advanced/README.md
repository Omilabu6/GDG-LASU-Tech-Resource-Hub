# Advanced Frontend Development

This section introduces deeper frontend topics that are commonly used in professional applications.

By this stage, learners should already understand:

- HTML
- CSS
- JavaScript
- DOM manipulation
- APIs
- React basics

The focus now is improving **code quality, performance, accessibility, and testing**.

---

# 1. Performance Optimization

Frontend performance is important for fast and smooth user experiences.

## What to Learn

- Lazy loading
- Code splitting
- Image optimization
- Minimizing bundle size
- Reducing render blocking resources

## Resources

Web Performance Guide  
https://web.dev/fast/

React Performance Optimization  
https://react.dev/learn

---

# 2. Accessibility (A11y)

Accessibility ensures websites can be used by everyone, including people with disabilities.

## What to Learn

- semantic HTML
- ARIA attributes
- keyboard navigation
- screen reader compatibility
- color contrast

## Resource

MDN Accessibility Guide  
https://developer.mozilla.org/en-US/docs/Web/Accessibility
