# Intermediate Frontend Development

This section introduces modern frontend development tools and frameworks used in real-world applications.

By this stage, learners should already understand:

- HTML
- CSS
- JavaScript fundamentals
- DOM manipulation
- Fetch API
- Async JavaScript

Now the focus shifts to building **larger, structured applications**.

---

# 1. Modern JavaScript (Deeper Understanding)

Before learning frameworks, it’s important to strengthen modern JavaScript knowledge.

## What to Learn

- ES6 modules (import/export)
- array methods (map, filter, reduce)
- destructuring
- spread and rest operators
- error handling
- code organization

## Primary Resource

JavaScript.info  
https://javascript.info

## Optional Resource

freeCodeCamp JavaScript Course  
https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/

---

# 2. Introduction to React

React is one of the most widely used libraries for building modern user interfaces.

## What to Learn

- components
- JSX
- props
- state
- event handling
- conditional rendering
- lists and keys

## Primary Resource

React Official Documentation  
https://react.dev

## Optional Resource

freeCodeCamp Frontend Libraries Course  
https://www.freecodecamp.org/learn/front-end-development-libraries/

## Important Hooks

- useState
- useEffect
- useContext

## Resource
