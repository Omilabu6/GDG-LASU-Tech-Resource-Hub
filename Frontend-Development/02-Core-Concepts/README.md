# Core Concepts

This section builds on the fundamentals and introduces the concepts needed to build **dynamic and real-world web applications**.

Here you will learn how JavaScript interacts with web pages, how to fetch data from APIs, and how modern JavaScript works.

---

# 1. The DOM (Document Object Model)

The DOM represents the structure of a web page and allows JavaScript to interact with HTML elements.

## What to Learn

- What the DOM is
- Selecting elements
- Changing text and styles
- Creating and removing elements
- Event listeners

## Primary Resource

MDN DOM Introduction  
https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model/Introduction

## Optional Resource

JavaScript DOM Crash Course  
https://www.youtube.com/watch?v=0ik6X4DJKCc

## Practice

Build small interactive features such as:

- Toggle dark mode
- Show/hide menu
- Interactive button counter

---

# 2. Events in JavaScript

Events allow web pages to respond to user interactions.

## What to Learn

- click events
- form submission
- keyboard events
- input events
- event listeners

## Primary Resource

MDN Events Guide  
https://developer.mozilla.org/en-US/docs/Learn/JavaScript/Building_blocks/Events

## Practice

Build a **form validation system** that checks:

- empty inputs
- correct email format
- password length

---

# 3. Modern JavaScript (ES6+)

Modern JavaScript introduced new syntax that makes code cleaner and more powerful.

## What to Learn

- let and const
- arrow functions
- template literals
- destructuring
- spread and rest operators
- array methods (map, filter, reduce)

## Primary Resource

JavaScript.info Modern JavaScript  
https://javascript.info/

## Practice

Rewrite older JavaScript code using ES6 features.

Example exercises:

- Convert functions to arrow functions
- Use map/filter on arrays
- Use destructuring on objects

---

Modern web apps often fetch data from external servers.

## What to Learn

- What APIs are
- Fetch API
- JSON data
- handling responses
- handling errors

## Primary Resource

MDN Fetch API

## Optional Resource

JavaScript Fetch Tutorial

## Practice

- Movie search app

# 5. Async JavaScript

Fetching data from servers requires asynchronous programming.

## What to Learn

- Promises
- async / await
- error handling with try/catch

## Primary Resource

Async JavaScript Guide  
https://javascript.info/async-await

## Practice

Modify your API project to use **async/await**.

# 6. Browser Storage

Browsers allow websites to store small amounts of data locally.

## What to Learn

- localStorage
- sessionStorage
- storing JSON data

## Primary Resource

MDN Web Storage API  
https://developer.mozilla.org/en-US/docs/Web/API/Web_Storage_API

## Practice

Upgrade your **To-Do List app** to save tasks using localStorage.
