# Practice

Learning frontend development requires **consistent practice**.

This section provides platforms and exercises that help reinforce what you learned in:

- Fundamentals
- Core Concepts

The goal is to **write code regularly and solve small challenges**.

---

# 1. HTML & CSS Practice

These platforms help you practice building layouts and user interfaces.

## Frontend Mentor

Frontend Mentor provides real-world UI challenges where you build designs from mockups.

https://www.frontendmentor.io

What you practice here:

- HTML structure
- CSS layouts
- responsive design
- recreating real UI designs

Recommended for beginners.

---

## CSSBattle

CSSBattle helps improve your CSS layout and creativity skills.

https://cssbattle.dev

Focus areas:

- positioning
- layout
- CSS problem solving

---

## Flexbox Practice

Flexbox Froggy  
https://flexboxfroggy.com

Helps you master **Flexbox layout** through interactive puzzles.

---

## CSS Grid Practice

CSS Grid Garden  
https://cssgridgarden.com

Helps you practice **CSS Grid layouts**.

---

# 2. JavaScript Practice

These platforms help improve your JavaScript problem-solving skills.

## Codewars

https://www.codewars.com

Recommended for:

- beginner JavaScript problems
- practicing functions and logic

Start with **8kyu and 7kyu problems**.

---

## HackerRank

https://www.hackerrank.com/domains/javascript

Helps practice:

- JavaScript fundamentals
- loops
- arrays
- problem solving

---

## LeetCode (Optional)

https://leetcode.com

Better for learners who want stronger algorithm and logic skills.

---

# 3. UI Recreation Practice

One of the best ways to improve is to **recreate real websites**.

Try recreating:

- Google homepage
- Netflix landing page
- Twitter profile page
- Apple product page
- GitHub homepage

Focus on:

- layout accuracy
- responsiveness
- clean HTML structure

---

# 4. Small Coding Challenges

Practice building small features like:

- responsive navigation bar
- image slider
- modal popup
- form validation
- dark mode toggle
- dropdown menu

These exercises strengthen **DOM manipulation and event handling**.

---

# Practice Tips

To get the most out of practice:

- Code every day, even if it’s for a short time
- Focus on understanding, not memorizing
- Build small features frequently
- Try recreating designs from scratch
