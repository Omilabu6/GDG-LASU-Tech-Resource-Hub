# Projects

Projects are the best way to strengthen your frontend development skills.

This section provides project ideas that help you apply what you learned from:

- Fundamentals
- Core Concepts
- Practice

These projects gradually increase in difficulty.

Try to **complete several of them and publish them on GitHub**.

---

# Beginner Projects

These projects help reinforce HTML, CSS, and basic JavaScript.

## Personal Portfolio Website

Build a website that introduces yourself.

Features:

- About section
- Skills section
- Projects section
- Contact form
- Responsive design

Skills practiced:

- HTML structure
- CSS layout
- Responsive design

---

## Responsive Landing Page

Create a modern landing page for a product or service.

Features:

- Navigation bar
- Hero section
- Features section
- Testimonials
- Footer

Skills practiced:

- Flexbox
- Grid
- responsive layouts

---

## Simple Calculator

Build a calculator that performs basic arithmetic.

Features:

- Addition
- Subtraction
- Multiplication
- Division

Skills practiced:

- JavaScript logic
- DOM manipulation
- event handling

---

## To-Do List App

Create a simple task manager.

Features:

- Add tasks
- Mark tasks as complete
- Delete tasks

Skills practiced:

- DOM manipulation
- event handling
- JavaScript functions

# Intermediate Projects

These projects introduce APIs and more dynamic functionality.

## Weather App

Create an app that shows weather information for a city.

Features:

- search city
- display temperature
- display weather conditions

Skills practiced:

- Fetch API
- async/await
- working with APIs

Suggested API:

https://openweathermap.org/api

---

## Movie Search App

Create a website that allows users to search for movies.

Features:

- search bar
- movie results
- movie details

Skills practiced:

- API integration
- handling JSON data
- dynamic UI updates

Suggested API:

https://www.omdbapi.com/

---

## Notes App (with Local Storage)

Create a notes application that saves data in the browser.

Features:

- create notes
- edit notes
- delete notes
- persistent storage

Skills practiced:

- localStorage
- DOM manipulation
- application logic

# Project Tips

When building projects:

- Start simple
- Focus on functionality first
- Then improve design
- Use Git for version control
- Write clean code
- Document your project
