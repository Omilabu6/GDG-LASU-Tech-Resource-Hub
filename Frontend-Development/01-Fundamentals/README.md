# Frontend Fundamentals

This section is designed for **beginners starting frontend development**.

Frontend development is built on three core technologies:

- **HTML** → Structure
- **CSS** → Styling
- **JavaScript** → Interactivity

The goal of this section is to help you understand these fundamentals and build your first simple web projects.

---

# 1. How the Web Works

Before writing code, it's important to understand how websites actually work.

## What to Learn

- What happens when you type a URL in the browser
- Client vs Server
- What a browser does
- HTTP and HTTPS
- How websites are hosted

## Primary Resource

MDN – How the Web Works  
https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/How_the_Web_works

## Optional Resource

How the Internet Works (Video)  
https://www.youtube.com/watch?v=7_LPdttKXPc

# 2. HTML — Structuring Web Pages

HTML (HyperText Markup Language) defines the structure of web pages.

## What to Learn

- HTML document structure
- Headings and paragraphs
- Links and images
- Lists
- Tables
- Forms
- Semantic HTML
- Navigation structure

## Primary Resource

MDN HTML Guide  
https://developer.mozilla.org/en-US/docs/Learn/HTML

## Optional Resource

freeCodeCamp Responsive Web Design  
https://www.freecodecamp.org/learn/2022/responsive-web-design/

## Practice

Build a simple webpage that includes:

- A title
- Navigation links
- Images
- A contact form

---

# 3. CSS — Styling Web Pages

CSS (Cascading Style Sheets) controls the appearance and layout of websites.

## What to Learn

- CSS syntax
- Selectors
- Colors and typography
- Box model
- Margin and padding
- Layout with Flexbox
- Layout with Grid
- Responsive design
- Media queries

## Primary Resource

MDN CSS Guide  
https://developer.mozilla.org/en-US/docs/Learn/CSS

## Optional Resources

Flexbox Guide  
https://css-tricks.com/snippets/css/a-guide-to-flexbox/

Grid Guide  
https://css-tricks.com/snippets/css/complete-guide-grid/

## Practice

Practice layout using:

Flexbox Froggy  
https://flexboxfroggy.com

CSS Grid Garden  
https://cssgridgarden.com

Mini Project:

Create a **responsive landing page**.

JavaScript makes websites interactive and dynamic.

## What to Learn

- Variables and data types
- Arrays and objects
- Functions
- Conditional statements
- Loops
- Basic debugging with console.log()

## Primary Resource

JavaScript.info  
https://javascript.info/

## Optional Resource

freeCodeCamp JavaScript Course  
https://www.freecodecamp.org/learn/javascript-algorithms-and-data-structures/
