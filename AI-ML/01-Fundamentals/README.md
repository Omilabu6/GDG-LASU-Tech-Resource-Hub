# AI / Machine Learning Fundamentals

This section introduces the basic concepts needed before learning machine learning.

---

# 1. What is Artificial Intelligence?

Artificial Intelligence refers to systems that simulate human intelligence to perform tasks such as:

- recognizing images
- understanding language
- making predictions
- learning from data

Resource

Introduction to AI  
https://www.ibm.com/topics/artificial-intelligence

---

# 2. What is Machine Learning?

Machine Learning is a subset of AI that allows computers to learn patterns from data.

Instead of being explicitly programmed, models improve as they see more data.

Resource

Machine Learning Explained  
https://www.ibm.com/topics/machine-learning

# 3. Python for AI

Python is the most common programming language used in AI.

Concepts to learn:

- variables
- loops
- functions
- lists and dictionaries
