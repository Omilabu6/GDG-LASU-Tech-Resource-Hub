# Core Concepts

This section introduces the key ideas behind machine learning models.

---

# 1. Types of Machine Learning

Machine learning is usually divided into three main categories.

Supervised Learning  
Models learn using labeled data.

Unsupervised Learning  
Models find patterns without labeled data.

Reinforcement Learning  
Agents learn by interacting with environments.

# 2. Machine Learning Workflow

A typical ML workflow includes:

1. collecting data
2. cleaning data
3. training a model
4. evaluating performance
5. improving the model

---

# 3. Common Algorithms

Examples include:

- Linear Regression
- Logistic Regression
- Decision Trees
- K-Nearest Neighbors
- Neural Networks

# 4. Model Evaluation

After training a model, it must be evaluated.

Common evaluation metrics include:

- accuracy
- precision
- recall
- F1 score
