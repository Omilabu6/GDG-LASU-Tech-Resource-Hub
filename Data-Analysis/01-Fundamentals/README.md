---

# 1. What is Data Analysis?

Data analysis is the process of inspecting, cleaning, transforming, and modeling data to extract meaningful insights.

Data analysts help organizations answer questions such as:

- What trends exist in our data?
- What factors influence performance?
- How can we improve outcomes?

# 2. Types of Data

Data can be categorized into different types.

Common types include:

- numerical data
- categorical data
- time-series data
- text data

Understanding data types helps determine the correct analysis techniques.
