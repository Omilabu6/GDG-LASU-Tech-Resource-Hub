# Core Concepts

This section introduces key techniques used in data analysis.

---

# 1. Data Cleaning

Real-world data is often messy.

Common cleaning tasks include:

- removing duplicates
- handling missing values
- correcting formatting errors

Resource

Pandas Data Cleaning  
https://pandas.pydata.org/docs

---

# 2. Exploratory Data Analysis (EDA)

EDA helps analysts understand datasets before modeling.

Techniques include:

- summary statistics
- distribution analysis
- correlation analysis

Resource

EDA Guide  
https://www.kaggle.com/learn/pandas

---

# 3. Data Visualization

Visualizations help communicate insights clearly.

Common charts include:

- bar charts
- line charts
- histograms
- scatter plots

Tools include:

- Matplotlib
- Seaborn
- Tableau
- Power BI

Resource

Data Visualization Guide  
https://matplotlib.org

---

# 4. SQL for Data Analysis

SQL allows analysts to retrieve and manipulate data stored in databases.

Important SQL concepts include:

- SELECT queries
- filtering data
- joins
- aggregations

Resource

SQL Tutorial  
https://www.w3schools.com/sql
