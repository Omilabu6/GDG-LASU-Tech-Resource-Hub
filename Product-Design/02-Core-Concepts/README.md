# Core Concepts

This section introduces key UI/UX design techniques used in real product development.

---

# 1. User Research

Understanding users is essential in product design.

Common research methods include:

- interviews
- surveys
- usability testing
- user observation

Resource

User Research Basics  
https://www.nngroup.com/articles/ux-research-cheat-sheet/

---

# 2. User Personas

Personas represent typical users of a product.

They help designers understand:

- user goals
- frustrations
- behaviors

---

# 3. Wireframing

Wireframes are simple layouts used to plan the structure of a product interface.

They focus on:

- layout
- content placement
- navigation

Tools include:

Figma  
https://figma.com

---

# 4. Prototyping

Prototypes simulate how a product will behave.

They allow designers to test ideas before development begins.

---

# 5. Usability Testing

Usability testing helps evaluate how easily users interact with a product.

Designers observe users performing tasks and identify issues.
