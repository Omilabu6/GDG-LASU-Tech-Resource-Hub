# Product Design Projects

Projects help strengthen your design thinking and build a strong portfolio.

---

# Beginner Projects

Mobile App Redesign

Choose an existing mobile app and redesign its interface.

Focus on:

- improving usability
- simplifying navigation
- improving layout

---

Website Landing Page Design

Design a landing page for a product or service.

Focus on:

- visual hierarchy
- call-to-action buttons
- clear messaging

---

# Intermediate Projects

Complete Mobile App Design

Design an entire mobile application including:

- onboarding
- main dashboard
- settings page

---

Product Dashboard Design

Design a dashboard used for analytics or management tools.

Focus on:

- data visualization
- layout structure
- usability
