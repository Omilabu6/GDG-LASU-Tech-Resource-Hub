# Product Design Fundamentals

This section introduces the basic concepts needed before designing digital products.

---

# 1. What is Product Design?

Product design focuses on creating solutions that meet user needs while achieving business goals.

Product designers focus on:

- usability
- accessibility
- functionality
- aesthetics

Resource

Introduction to Product Design  
https://www.interaction-design.org/literature/topics/product-design

---

# 2. UI vs UX Design

UI (User Interface)

Focuses on the visual elements of a product such as:

- buttons
- colors
- typography
- layouts

UX (User Experience)

Focuses on how users interact with the product.

Examples include:

- navigation flow
- usability
- user satisfaction

---

# 3. Design Thinking

Design thinking is a problem-solving approach used in product design.

Steps include:

1. Empathize with users
2. Define the problem
3. Generate ideas
4. Build prototypes
5. Test solutions

Resource

Design Thinking Guide  
https://www.ideou.com/blogs/inspiration/design-thinking

---

# 4. Basic Design Principles

Important principles include:

- visual hierarchy
- contrast
- alignment
- spacing
- consistency

---

# First Exercise

Try analyzing an app you use daily.

Ask questions such as:

- Why is the layout designed this way?
- How easy is navigation?
- What could be improved?
